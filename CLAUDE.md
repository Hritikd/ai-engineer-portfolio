# Portfolio Project

## What this is
A personal portfolio website for Hritik Datta, showcasing my AI engineering learning journey and projects.

## About the owner
- Learning AI engineering: Claude Code, MCP, tool use, agents
- Background: Python and JavaScript basics
- Goal: build publicly, show recruiters real AI projects

## Tech rules
- HTML, CSS, and vanilla JavaScript only (no frameworks, no React)
- Mobile-responsive using CSS Grid and Flexbox
- Clean, minimal, professional design
- All code must have comments explaining what it does
- Semantic HTML (use header, main, section, footer, nav — not divs for everything)

## Structure
- Single page site (index.html, style.css, script.js)
- Sections: Hero (name + tagline), About Me, Projects (3 cards), Skills, Contact
- Projects section will link to GitHub repos as I build them

## Deployment
- GitHub Pages (free, auto-deploys from main branch)
